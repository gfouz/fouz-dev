Projects and their Links

Technology blog.

This blog is about latest web innovations. And this blog itself is built with Next.Js using the app directory and server components, a new type of React components that run on the server and return compiled JSX that is sent to the client. Server Components are useful for rendering the skeleton of a page, fetching data from the server in parallel, and passing it to the "client components".

Finances tracker.

Finances tracker is a dynamic UI that is built with Next.Js and Zustand to create a global state for managing and persisting data within its components that can capture and respond to user input, such as clicks, keyboard events, and form submissions. Global state can change over time and impact how the components appears or behaves during their life cycle. This project is useful for tracking personal finances. I will build a finances tracker for desktop with new versions of Nextron.
